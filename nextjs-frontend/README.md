## Commands

```bash
npx create-next-app@latest --ts
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
npm install react-leaflet
npm install -D @types/leaflet
npm install zod
npm install leaflet-defaulticon-compatibility
```
